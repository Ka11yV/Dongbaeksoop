## ğŸš€ Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/Dongbaeksoop.git
cd Dongbaeksoop
```

### 2. Install Dependencies & Build CSS
```bash
# Install Tailwind CSS dependencies
npm install

# Build CSS (Watch mode for development)
npx tailwindcss -i ./src/main/webapp/styles/input.css -o ./src/main/webapp/styles/output.css --watch
```

### 3. Run with Maven
```bash
./mvnw clean package
# Deploy the generated WAR file to your Tomcat server
```
