# 동백숲 (Dongbaeksoop)

## 프로젝트 정의 및 목표
- 프로젝트 정의 : 동양미래대 학생이 직접 각 강의에 대해 평가(별점)을 남길 수 있고, 학교 공지 사항을
메일로 받을 수 있는 기능을 제공하는 웹 서비스
- 사용자 대상 : 동양미래대 학생
- 프로젝트 이름 : 동백숲

## 배경 & 목적
- 배경
  - 강의 평가 : 수강신청을 할 때 마다 자신에게 맞는 수업인가에 대한 고민
  - 공지 메일 : 학교 공지사항을 확인하려면 자주 학교 웹사이트에 들어가야 하는 번거로움
- 목적
  - 강의 평가 : 자신에게 적합한 강의를 찾고 다른 학우들에게 자신의 강의평가를 들려줌
  - 공지 메일 : 학교 공지사항 중에서 자신이 키워드를 설정하여 원하는 공지를 메일로 받도록 함
